# Bank Loan Analysis Dashboard

An end-to-end Business Analyst project focused on analyzing customer loan data using SQL, Power BI, Tableau, and Excel.

This project includes:
- Interactive dashboards
- KPI reporting
- Loan performance analysis
- State-wise and customer-wise insights
- Data visualization and reporting

## Tools Used
- SQL
- Power BI
- Tableau
- Advanced Excel

## Key KPIs
- Year-wise Loan Amount Stats
- Grade and Sub-grade Analysis

- Verified vs Non-Verified Payments
- State-wise Loan Status
- Home Ownership Analysis

## Skills Demonstrated
- Data Cleaning
- Data Visualization
- Dashboard Creation
- KPI Reporting
- SQL Query Writing
- Business Insights Generation
